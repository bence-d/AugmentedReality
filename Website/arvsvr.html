Smart Glasses

Augmented Reality Glasses or also called Smart Glasses: Smart Glasses capture the environment, by scanning them using a camera. They identify points where the Ar can overlay an 3D Object. It can find the positioning using either a marker or a tracker like for example a infrared, a laser, a sensor or an GPS. So yes with those Smart Glasses you see the Objects added with Ar.
